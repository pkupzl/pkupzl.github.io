前言

在阅读串的KMP算法的时候，我竟然没搞清楚下标到底是从0开始还是从1开始


在此写一个从0开始的笔记

第一章 next数组

next数组的定义

$$ next[i]=\begin{cases}
-1,\quad i=0
max{k}, \quad s.t“t1t2…tk-1“=”tj-k…tj-1“
\end{cases} $$

next数组的求法

int getnext()
{
    int i,j;
    i=0; next[i]=-1;
    j=-1;
    while(i<l2)
    {
        if(j==-1||s2[i]==s2[j])
        {
            i++;j++;
            next[i]=j;    
        }
        else
        {    
        j=next[j];
        }    
    }
}

下面我们从递推的角度思考这个算法


假设next[i]=j

  1. ti=tj

    那么j$\rightnarrow$j+1
    i$\rightnarrow$i+1
    next[i]=j
  2. ti!=tj

    那么令j=next[j] (迭代)

    若next[i+1]>j+1 则ti=tj，矛盾

    所以next[i+1]<=j+1

所以这样就保证了答案（next[i+1]）永远在j指针+1的左边，而在迭代过程中j保持递减（到-1结束）故我们总是能找到
